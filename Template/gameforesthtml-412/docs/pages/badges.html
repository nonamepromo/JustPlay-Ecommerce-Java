Badges

Documentation and examples for badges, our small count and labeling component. Badges scale to match the size of the immediate parent element by using relative font sizing and em units.

Example

Add any of the below mentioned modifier classes to change the appearance of a badge.Using color to add meaning only provides a visual indication, which will not be conveyed to users of assistive technologies â€“ such as screen readers.

  • Preview
  • Markup
Primary Success Info Warning Danger Dark 
<span class="badge badge-primary">Primary</span>
<span class="badge badge-success">Success</span>
<span class="badge badge-info">Info</span>
<span class="badge badge-warning">Warning</span>
<span class="badge badge-danger">Danger</span>
<span class="badge badge-dark">Dark</span>

Outline

Badges can be used with outline style just add .badge-outline-* to badge class. It can be good solution to show visitors a custom rank like administrator with primary color.

  • Preview
  • Markup
Primary Success Info Warning Danger Dark 
<span class="badge badge-outline-primary">Primary</span>
<span class="badge badge-outline-success">Success</span>
<span class="badge badge-outline-info">Info</span>
<span class="badge badge-outline-warning">Warning</span>
<span class="badge badge-outline-danger">Danger</span>
<span class="badge badge-outline-dark">Dark</span>

Category

Display categories with category badges and add .badge-category to badge class. Easy to modify custom category colors with scss in variables.scss file.
