Accordions

The Gameforest collapse plugin allows you to toggle content on your pages with a few classes thanks to some helpful JavaScript.

Example

Extend the default collapse behavior to create an accordion with the card component. Be sure to add aria-expanded to the control element. The plugin will automatically toggle this attribute based on whether or not the collapsible element.
